## 言語処理系（５）_p005
<!-- source: 言語処理系（５）.pdf page 5 -->

int foo(int x) {
int y = x * x;
return y + 2;
}
_foo: push ebp
mov  ebp, esp
sub  esp, 4
mov  eax, 8[ebp]
imul eax, 8[ebp]
mov  -4[ebp], eax
mov  eax, -4[ebp]
add  eax, 2
mov  esp, ebp
pop  ebp
ret
５．１覗き穴式最適化
• 冗長命令の削除２
mov -4[ebp], eax
mov eax, -4[ebp]
mov eax, [ebx]
mov eax, [ebx]
のそれぞれにおいて，２つ目の命令は冗長

## 言語処理系（５）_p021
<!-- source: 言語処理系（５）.pdf page 21 -->

５．４レジスタ変数
• 変数の参照回数
– 代入１回，参照１回
• ループ中の参照：ループをK回として推定
for (j = i; j < n; j++) s += j;
j：4K+2回，s：2K回，n：K+1回，i：1回
• 条件文中の参照：確率を均等に割振り推定
if ( e ) s1 else s2;
(eでの参照回数)+((s1での参照回数)+(s2での参照回数)) / 2

![page image](../../images/pages/5_p021.png)
